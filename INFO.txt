NAME
        BLUEENGINE - Opensource aquifer level analysis, processing, predictions tool

SYNOPSIS
        WINDOWS [CMD/POWERSHELL]
        python blue.py -[FLAGS] --[ACTIONS]
        LINUX [SH/BASH]
        python3 blue.py -[FLAGS] --[ACTIONS]

DESCRIPTION
        BLUEENGINE is written as a python cli tool to handle data from various sources.
        It helps with it's preprocessing, modelling, prediction making, analysis etc.
        To use it, use python to run the blue.py with additional flags preceded by '-'
        and '--' for actions

        Multiple actions can be chained e.g. --help --setup
        Flags can be written in a single arguement e.g. -abc

OPTIONS
        Generic Program Information
            --help Output a usage message and exit

        --version
                Output the version number of BLKSIG and exit

        Repetition
                A regular expression may be followed by one of several repetition operators:
                ?      The preceding item is optional and matched at most once.
                *      The preceding item will be matched zero or more times.
                +      The preceding item will be matched one or more times.
                {n}    The preceding item is matched exactly n times.
                {n,}   The preceding item is matched n or more times.
                {,m}   The preceding item is matched at most m times. This is a GNU extension.
                {n,m}  The preceding item is matched at least n times, but not more than m times.

        Ctrl+c
